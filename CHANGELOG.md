# CHANGELOG

## v2 - 2026-06-14

### 追加
- 音声ダウンロードの品質プリセットを追加し、設定画面とダウンロード画面で選択できるようにしました。
- ライブラリで全選択、選択解除、選択削除、件数表示を追加しました。
- サムネイル画像を縮小デコードしてキャッシュするコンバーターを追加しました。
- ビルド後に指定フォルダーへ成果物をコピーする設定を追加しました。
- 処理フローと評価・比較資料を追加しました。

### 変更
- mp3/wav変換時にffmpegの進捗を表示し、変換・メタデータ書き込みなどの後処理ステータスを表示するようにしました。
- m4aではAAC音源を優先し、再エンコードを避けやすい形式選択へ変更しました。
- 設定保存後の既定フォーマット・品質を次回URL解析時に反映するようにしました。
- ライブラリ検索を入力デバウンス付きのライブ検索へ変更し、一覧の再読み込みを直列化しました。
- ライブラリ一覧の仮想化とサムネイル読み込みを改善しました。
- ナビゲーション選択の同期方式を見直し、ライブラリ表示時の読み込みを確実に実行するようにしました。
- yt-dlpの進捗解析とJSONプロパティ取得をより堅牢にしました。

### 修正
- ダウンロードのキャンセル時にyt-dlp/ffmpegプロセスが残る問題を修正しました。
- 待機中のジョブをキャンセルしてもセマフォ取得後に実行される問題を修正しました。
- 保存先フォルダーを作成できない場合にアプリがクラッシュしないようにしました。
